NEXOVA Digital — Structure du site
===================================

Fichiers :
- index.html      → Page d'accueil (hero, services, portfolio preview, stats, témoignages)
- about.html      → L'agence (histoire, équipe, valeurs, certifications)
- services.html   → Services détaillés + tarifs + FAQ
- portfolio.html  → Réalisations + études de cas + filtres
- blog.html       → Articles + sidebar newsletter + catégories
- contact.html    → Formulaire intelligent + infos + FAQ
- style.css       → Toute la charte graphique (bleu/violet/noir/or + glassmorphism)
- galaxy.js       → Animation galaxie canvas + interactions globales

Pour personnaliser :
- Nom de l'agence : chercher "NEXOVA" dans tous les fichiers et remplacer
- Couleurs : modifier les variables CSS dans :root { } au début de style.css
- Contenu : modifier directement chaque fichier HTML
- Formulaire contact : brancher contact.html à un backend (ex: EmailJS, Formspree, etc.)
